Here are 10 slightly harder Python problems:


1. Prime number checker
Ask the user for a number and check if it is a prime number or not.




2. Find all primes up to N
Take a number N and print all prime numbers from 1 to N.




3. Simple calculator (functions)
Create a calculator using functions:
add
subtract
multiply
divide
Ask user for choice.




4. Count digits in a number
Ask user for a number and count how many digits it has.
Example:
Input: 12345
Output: 5



5. Palindrome checker
Check if a word or number is a palindrome (same forward and backward).
Example:
madam → Yes
hello → No



6. Find largest number in a list
Take a list of numbers from user and find the largest without using max().




7. Reverse a list manually
Take a list from user and reverse it without using [::-1].



8. Fibonacci series
Print first n Fibonacci numbers.
Example (n=6):
0 1 1 2 3 5




9. Remove duplicates from a list
Take a list and remove repeated values.
Example:
Input: [1,2,2,3,3,3]
Output: [1,2,3]





10. Word frequency counter
Take a sentence and count how many times each word appears.
Example:
Input: I love python and I love coding
Output:
I → 2
love → 2
python → 1
and → 1
coding → 1
